  • 配置你的VN工程

    使用VSCode时在工程目录下添加一个jsconfig.json文件，具体可以参考如下

    {
// See https://go.microsoft.com/fwlink/?LinkId=759670
// for the documentation about the jsconfig.json format
"compilerOptions": {
 "target": "es5",
 "module": "commonjs",
 "allowSyntheticDefaultImports": true
},
"exclude": [
 "node_modules",
 "bower_components",
 "jspm_packages",
 "tmp",
 "temp"
]
}

    在工程目录下添加VN的d.ts文件即可
